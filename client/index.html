Timestamp microservice - by Jeff "Deftwun" Wakeman

Pass either a natural language date or unix timestamp as a parameter to this url.

Server will respond with both formats in JSON format.

Example Input

  • https://thawing-escarpment-1689.herokuapp.com/December%2015,%202015
  • https://thawing-escarpment-1689.herokuapp.com/1450137600

Example Output

  • { "unix": 1450137600, "natural": "December 15, 2015" }
